El proposito de este sitio web es mostrar todo el contenido que se requiere para el desarrollo de las diferentes etiquetas, atributos y valores que manejamos en HTML, para que todos puedan tener en cuenta la importancia que esto conlleva, el fin seria mostrarle a todos los interesados en el diseño de los sitios web, la importacia de aprende a manejar estas herramientas que se nos permita ejecutar.

Atributos y valores: Básicamente, la sintaxis de HTML consiste en etiquetas, atributos y los elementos que conforman la página, tanto el texto como las imágenes, tablas… hacerlo, cosa que puede ocurrir con los valores de algunos atributos (por ejemplo, nombres de ﬁcheros).
